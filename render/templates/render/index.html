Programmed Differently?

Brooke,S. (2023). Testing for Gender Differences in Python Programming Style and Quality on GitHub. Journal of Computer-Mediated Communication. https://doi.org/10.1093/jcmc/zmad049

Abstract.The underrepresentation of women in open-source software is frequently attributed to women’s lack of innate aptitude compared to men: a natural “gender difference” in technical ability. Approaching code as a form of communication, I conduct a novel empirical study of gender difference in Python programming on GitHub. Based on 1,728 open-source projects, I ask if there is a gender difference in the quality and style of Python code measured in adherence to PEP-8 guidelines. I found significant gender differences in structure and how Python files are organised. Whilst there is gendered variation in programming style, there is no evidence of gender difference in code quality. Using a Random Forest model, I show that the gender of a programmer can be predicted from the style of their Python code. The study concludes that gender differences in Python code are a matter of style, not quality.

The purpose of this app is to allow you to interact with the findings and run the classification model on your own Python files.

This app will tell you if your Python code is written in a Masculine or Feminine style. This is not an attempt ito infer your gender. After you upload you file, you can click submit and you will see a score rating of your file, as well as a breakdown of the gendered style of your Python code as a percentage.
